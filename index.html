Features

Todo lo que necesitas para configurar tu avatar en un solo paquete.

🎬

Radial Animations

Genera animaciones OnOff, AB y Linear automaticamente a partir de frames capturados visualmente.

🛠

Non-Destructive (NDMF)

7 plugins NDMF aplican cambios solo en build. Tu avatar original nunca se modifica.

🪡

Bone Stitching

Cose armatures de ropa automaticamente con 230+ patrones de huesos compatibles.

🎨

Alternative Materials

Sistema de 3 capas para cambiar materiales con animaciones radiales lineales.

💡

Radial Illumination

Controla la iluminacion de shaders lilToon via menus radiales en VRChat.

⚖

PhysBone Organization

Reorganiza VRCPhysBones en contenedores limpios automaticamente.

📊

Texture Analysis

Analiza peso VRAM y Mip Streaming de texturas para optimizar tu avatar.

🗛

Bounds Adjustment

Unifica bounds de meshes para evitar culling visual en VRChat.

🌐

Multi-Language

Interfaz localizada en 6 idiomas: ES, EN, ZH, JA, KO, RU.

Installation

Agrega Menu Radial a tu proyecto en menos de un minuto.

1

Quick Install

Haz click en "Add to VCC" arriba. Esto abrira VRChat Creator Companion y agregara el repositorio automaticamente.

2

Manual Install

Si prefieres hacerlo manual: abre VCC, ve a Settings → Packages → Add Repository y pega la URL del listing.

3

Add to Project

Una vez agregado el repositorio, busca Menu Radial en la lista de paquetes de tu proyecto y haz click en "Add".
