# Bender_Dios - VPM Listing

Repositorio VPM para [VRChat Creator Companion](https://vcc.docs.vrchat.com/).

**Web**: https://emerytheec.github.io/vpm-listing/

## Quick Install

Haz click para agregar automaticamente a VCC:

[Add to VCC](vcc://vpm/addRepo?url=https://emerytheec.github.io/vpm-listing/index.json)

## Manual Install

1. Abre VRChat Creator Companion
2. Ve a **Settings** > **Packages** > **Add Repository**
3. Pega esta URL:
   ```
   https://emerytheec.github.io/vpm-listing/index.json
   ```
4. Click en **Add**
5. Busca los paquetes en la lista de tu proyecto

## Paquetes Disponibles

| Paquete | ID | Tipo | Descripcion |
|---------|----|------|-------------|
| Menu Radial | `com.bender-dios.menu-radial` | Avatares | Sistema automatizado para crear menus de expresiones radiales |
| Idiomas | `com.benderdios.idiomas` | Mundos | Sistema de localizacion para mundos VRChat (11 idiomas, UdonSharp) |
